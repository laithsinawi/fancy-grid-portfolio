Fancy Grid Portfolio

WordPress plugin to create portfolio in nice grid format that is animated and filterable with beautiful hover overlay with project title and description.

View on GitHub Download .zip Download .tar.gz

Description

Fancy Grid Portfolio is a WordPress portfolio grid plugin built to help you display your projects on a portfolio page in a nice grid format, and also works well for showing clients or team members. It includes a filter feature that allows you to create categories that sort and reorders your portfolio items with cool animation. The UI is designed to be super simple and fast to set up a portfolio without having to set tons of options.

This plugin uses a custom post type "portfolio_item" and custom taxonomy "portfolio_categories" to not clutter up your normal posts categories and to keep things organized. It also takes advantage of WordPress Featured Image, and has an ajax drag drop feature to help you quickly reorder your portfolio items.

Plugin Features

  • Responsive Portfolio Grid
  • Category Filtering with Animation
  • Beautiful Overlay on Hover with Project Title and Description
  • Lightbox Popups
  • Drag/Drop Sort to Quickly Reorder Portfolio Items
  • Shortcode Support
  • Set Crop Width and Height
  • Set Number of Columns - 2, 3, or 4 Columns
  • Simple and Intuitive Interface for Quick Portfolio Setup

Installation

  1. Install it as a regular WordPress plugin

  2. Add your portfolio posts with title, description, image and categories

  3. Then use the following shortcode to display portfolio

[fgp_portfolio]

Frequently Asked Questions

How can I change portfolio number of column?

Go to Grid Portfolio -> Dashboard and look under Plugin Options section for Number of Columns setting.

Why is my portfolio grid showing different size images?

You should upload images and not use existing ones from the media library for your portfolio in order for the crop to work. Then your images will all be the same size and the grid will be consistent.
